ESP32-S3-WROOM-1 V1.0 — Custom PCB
A custom-designed ESP32-S3 board with onboard I²C sensors (LIS2MDL, BME280) and multiple GPIO breakout headers. Designed in KiCad, fabricated via JLCPCB. Rev-1.0
